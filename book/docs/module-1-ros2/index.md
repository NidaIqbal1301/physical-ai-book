# Module 1: The Robotic Nervous System (ROS 2)

This module covers the fundamentals of ROS 2, the middleware that serves as the nervous system for robot control. You'll learn how to connect digital AI agents with physical robots using ROS 2 architecture.

## Topics Covered

- ROS 2 architecture: nodes, topics, services, actions
- Bridging Python agents to ROS controllers (rclpy)
- Humanoid description formats (URDF)

## Learning Objectives

- Understand ROS 2 fundamentals
- Control robot hardware/software interfaces
