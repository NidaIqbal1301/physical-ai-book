# Humanoid Description Formats (URDF)

This chapter covers the Unified Robot Description Format (URDF) used to describe humanoid robots in ROS 2 systems.

## URDF Components

- Links: Rigid parts of the robot (e.g., arms, legs, torso)
- Joints: Connections between links (e.g., hinges, prismatic joints)
- Visual and collision properties
- Inertial properties

## Learning Objectives

- Create URDF files for humanoid robots
- Understand the structure of robot descriptions
- Apply URDF to simulate and control humanoid robots
