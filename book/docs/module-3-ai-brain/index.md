# Module 3: The AI-Robot Brain (NVIDIA Isaac)

This module covers advanced perception and AI training for humanoid robots using NVIDIA Isaac. You'll learn about Isaac Sim, Isaac ROS packages, and navigation systems.

## Topics Covered

- Isaac Sim: photorealistic simulation, synthetic data generation
- Isaac ROS: VSLAM, navigation, path planning
- Nav2 for bipedal humanoid movement

## Learning Objectives

- Train AI for perception, navigation, and humanoid movement
