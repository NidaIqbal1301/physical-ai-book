# Module 2: The Digital Twin (Gazebo & Unity)

This module focuses on physics simulation and environment building for humanoid robots. You'll learn about physics simulation, high-fidelity rendering, and sensor simulation for creating digital twins.

## Topics Covered

- Physics simulation: gravity, collisions, dynamics
- High-fidelity rendering and human-robot interaction (HRI) in Unity
- Sensor simulation: LiDAR, Depth Cameras, IMUs

## Learning Objectives

- Build and simulate digital twins for humanoid robots
